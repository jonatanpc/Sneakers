    Térmogenico

    Son activadores metabólicos porque activan el sistema simpático adrenal. Este sistema simpático adrenal se activa también para producir energía inmediata en situaciones de estrés o de lucha. Al activar este sistema, ganamos energía, pero además a nivel metabólico, se activa la lipólisis, es decir, “la producción de energía a partir de las grasas que se encuentran en los adipocitos”.
